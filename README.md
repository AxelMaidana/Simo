# SIMO — Leasing de motos para trabajar

Landing page construida con [Astro](https://astro.build), [Tailwind CSS](https://tailwindcss.com), [GSAP](https://gsap.com) y [Lenis](https://lenis.darkroom.engineering) para el smooth scroll.

## Desarrollo

```sh
npm install
npm run dev
```

## Build de producción

```sh
npm run build
npm run preview
```

## Deploy en Netlify

El repo incluye `netlify.toml` con la configuración de build (`npm run build`, carpeta publicada `dist`). Solo hace falta conectar el repositorio en Netlify.
